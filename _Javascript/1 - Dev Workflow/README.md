## Advice

- Pace yourself
  - Practice
- Ask questions
- Ask Google
- Seek to understand why or how?
  - Developers are problem solvers
  - Step by step incremental development
- Conceptualize the solution or process before you code. Pseudocode etc
- Keep going and never give up
